    # 通过阴影弱化背景

    # 常规做法

    .overlay{
   position:fixed;
   top:0;
   right:0;
   bottom:0;
   top:0;
   background:rgba(0,0,0,.7)
}
.lingthbox{
  position:absolute;
  z-index:1
  /* 其他元素 */
}

    # body上加伪元素（不推荐）

    body.dimmed::before{
   position:fixed;
   top:0;
   right:0;
   bottom:0;
   top:0;
    background:rgba(0,0,0,.7);
    z-index:1;

}

    缺点：

    1. body上可能有其他元素占用before，会被覆盖
    2. 伪元素无法绑定js事件处理函数

    # box-shadow方案

    .lightbox {
	position: fixed;
	top: 50%; left: 50%;
	margin: -200px;
	box-shadow: 0 0 0 50vmax rgba(0,0,0,.8);
}

    缺点： 1.鼠标可以继续点击页面做交互 2.页面如果有滚动则边缘会显露出来
