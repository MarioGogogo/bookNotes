# 3D旋转

# 实现思路:

就是将两个要展示的div利用定位重叠在一起，其中一个首先围绕Y轴旋转一定角度，还要加上一个关键属性backface-visibility:hidden (该属性定义当元素不面向屏幕时是否可见这个非常重要）。如果不加上这个属性的话，那么就会始终只能看到一张卡片了，失败的效果如下： html结构

//html结构
 <div class="item">
          <div class="front">
              <img src="front.png" alt="">
              <p>我是正面</p>
          </div>
          <div class="back">
              <img src="back.jpeg" alt="">
              <p>我是反面</p>
          </div>
     </div>

css结构

.item{
            width: 220px;
            margin: 50px auto;
            height: 276px;
            cursor: pointer;
            position: relative;
            perspective: 500px;
        }
        .item img{
            max-width: 220px;
            width: 100%;
            height: 150px;
        }
       .item .front,.item .back{
          position: absolute;
           left:0;
           top:0;
           width: 100%;
           height: 100%;
           background: #fff;
           overflow: hidden;
           perspective: 1000px;
           box-shadow: rgba(50, 50, 50, 0.8) 0 0 15px;
           transition: all 1.5s;
           backface-visibility:hidden  /*(该属性定义当元素不面向屏幕时是否可见）*/
       }
        /*先让back选择-180deg*/
        .back{
            position: relative;
            transform: rotateY(-180deg);
        }
        /*鼠标放上去去 前面那张图片选择 -180deg*/
        .item:hover .front{
            transform: rotateY(-180deg);
        }

        .item:hover .back{
            transform: rotateY(-360deg);
        }
