# 数组的缺点

数组作为最最常用的数据结构，应用范围很广，但它也有缺点。数组的大小是固定的，从数组的起点或者中间插入或移除某一个项成本很高,因为需要移动元素

# 什么是链表

链表是可以存储元素的集合,链表中的元素在内存中不是连续放置的(与数组的区别),每一个元素由一个储存的元素本身和一个指向下一个元素的指针组成。 链表的好处：添加或者移除元素的时候不需要移动元素。我们在查找某个元素的时候，链表的作法是从表头开始迭代链表直到找到所需元素；

push

  push(element) {
    const node = new Node(element);
    let current;
    if (this.head == null) {
      this.head = node;
    } else {
      current = this.head;
      while (current.next != null) {
        current = current.next;
      }
      current.next = node;
    }
    this.count++;
  }

  removeAt(index) {
    if (index >= 0 && index < this.count) {
      let current = this.head;
      if (index === 0) {
        this.head = current.next;
      } else {
        let previous;
        for (let i = 0; i < index; i++) {
          previous = current;
          current = current.next;
        }
        previous.next = current.next;
      }
      this.count--;
      return current.element;
    } else {
      return undefined;
    }
  }
