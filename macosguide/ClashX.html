# ClashX

提示

新一代代理工具，拥有更加先进的功能，适合高玩

# 软件下载

点击这里进入Github 项目页，下载 dmg 文件。

# 对比传统 SSR 优势

1.拥有自动选择/切换节点功能；

2.自带的规则对比 SSR 的 PAC 或自带规则更为完善，减少误判（特别对于 MAC 的 SSR 根本不能使用自动规则）；

3.可针对不同网站自动选择不同节点；

4.可以使用 Adblock 规则去广告（默认已开启）。

# 配置

1.前往订阅中心获取托管链接。

2.开启 ClashX，点击上方任务栏中的 ClashX -> Config -> Remote config -> Set URL，粘贴链接并确认

3.找到 Proxy 选项，在此处您可以选择节点。Auto-UrlTest 即代表自动选择最佳。如您有自己的使用习惯，也可自行选择节点。

下方还有很多选项，包括 Domestic，Others，Adblock 等等。这些属于策略组，如果您感兴趣，可以点击这里查看相关教程。请勿在看教程之前随意修改，否则可能出现问题！

规则名称 代表网站 默认选项
Proxy 被墙的网站 Auto-Urltest
Domestic 国内网站 Direct（直连）
AdBlock 广告网站 Reject（拒绝）
AsianTV 亚洲流媒体 Direct（直连）
GlobalTV 全球流媒体（如 NF) Proxy（代理）
Others 除以上列表的其它网站 Proxy（代理）

4.开启 Start at login 和 Set as system Proxy，实现开机自启并自动设置为系统代理

5.之后您可以在任务栏中切换代理节点或关闭代理
